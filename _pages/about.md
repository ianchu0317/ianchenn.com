If I had to describe myself with one word, it would be _curious_. 

Since I was a kid I like to learn and try new things, and so I'm very self-taught. 
A lot of skills I have came from watching videos online and practicing by myself, and I think I'm really good at it.

I've built this website to post my learnings and document my projects :)
